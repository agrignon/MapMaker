### Broken E2E Flows

**Generate Preview includes Overture buildings:** Breaks at fetch trigger. User clicks Generate → only OSM data is fetched → no Overture tiles retrieved → no gap-fill buildings rendered.

## Interface Readiness (Phase 10 → Downstream)

Phase 10's exports are well-structured for downstream consumption:

- `OvertureResult { tiles: Map<string, ArrayBuffer>, available: boolean }` — correct shape for Phase 11 decoder
- `OVERTURE_BUILDING_LAYER = 'building'` — exported constant ready for MVT layer selection
- `OVERTURE_FETCH_ZOOM = 14` — available for tile-bound computation
- `overtureAvailable` flag — lifecycle resets on `setBbox`/`clearBbox` ready for Phase 13

Key note for Phase 11: MVT geometries are tile-local coordinates (0-4096 grid) requiring WGS84 projection before constructing `BuildingFeature` instances.
