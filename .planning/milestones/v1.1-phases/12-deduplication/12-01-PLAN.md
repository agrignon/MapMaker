<feature>
  <name>deduplicateOverture — AABB IoU building deduplication</name>
  <files>src/lib/overture/dedup.ts, src/lib/overture/__tests__/dedup.test.ts</files>
  <behavior>
    deduplicateOverture(osmFeatures, overtureFeatures) => BuildingFeature[]:

    - Identical buildings (IoU = 1.0) → Overture removed
    - No overlap (IoU = 0.0) → Overture passes through as gap-fill
    - Partial overlap below threshold (IoU ~ 0.2) → Overture passes through
    - Partial overlap at threshold (IoU = 0.3) → Overture removed (>= not >)
    - Partial overlap above threshold (IoU = 0.5) → Overture removed
    - Empty OSM list → all Overture features returned unchanged
    - Empty Overture list → returns empty array
    - Multiple OSM buildings; Overture overlaps one → Overture removed
    - Multiple Overture buildings; only some overlap OSM → subset returned
    - Slightly misaligned duplicate (realistic OSM vs Overture offset) → removed
    - Degenerate AABB (zero-area bbox) → returns 0 IoU, no NaN
  </behavior>
  <implementation>
    Algorithm (from 12-RESEARCH.md, all plain TypeScript arithmetic):

    1. AABB extraction: For each building's outerRing, compute { minLon, maxLon, minLat, maxLat } using Math.min/Math.max over [lon, lat] pairs.

    2. AABB IoU: Given two AABBs, compute intersection rectangle. If no intersection, return 0. Otherwise: interArea / (areaA + areaB - interArea). Guard unionArea <= 0 → return 0 (degenerate case).

    3. Main function: Pre-compute OSM AABBs once. For each Overture building, compute its AABB and check if any OSM AABB produces IoU >= 0.3. Keep only Overture buildings that do NOT match any OSM building.

    Key constraints:
    - IoU threshold is EXACTLY 0.3, using >= (not strict >). This is a locked decision from STATE.md.
    - outerRing coordinates are [lon, lat] order (index 0 = longitude, index 1 = latitude).
    - Do NOT filter or modify OSM features. Only filter Overture features.
    - Do NOT merge/concat results. Return ONLY the filtered Overture gap-fill list. Phase 13 handles merging.
    - Do NOT mutate input arrays. Return a new filtered array.
    - No new npm dependencies. Pure TypeScript arithmetic.
    - Export DEDUP_IOU_THRESHOLD constant for test use.
  </implementation>
</feature>
