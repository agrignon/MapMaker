1. Create `src/lib/buildings/obb.ts`:
   - `computeOBB(vertices: [number, number][]): { center: [number, number]; halfExtents: [number, number]; axes: [[number, number], [number, number]] }`
   - Compute the 2D Oriented Bounding Box of a polygon using rotating calipers on the convex hull
   - Algorithm: (a) compute convex hull of the vertices, (b) for each edge of the convex hull, rotate coordinate system to align with that edge, (c) compute axis-aligned bounding box in rotated space, (d) keep the rotation that produces the minimum area
   - `axes[0]` = long axis direction (unit vector), `axes[1]` = short axis direction
   - `halfExtents[0]` = half-length along long axis, `halfExtents[1]` = half-length along short axis
   - This is ~60-80 lines of custom geometry code. No external library needed.

2. Extend `src/lib/buildings/roof.ts` with three new roof builders:

   a. `buildGabledRoof(topRingXY, topZmm, holes, roofHeightMM)`:
      - Compute OBB of the footprint using `computeOBB(topRingXY)`
      - Ridge line runs along the OBB long axis at Z = max(topZmm) + roofHeightMM, centered on the footprint
      - Ridge endpoints: OBB center +/- halfExtents[0] * axes[0]
      - For each wall edge: if it's on the "left side" of the ridge (dot product with short axis < 0), connect to the ridge line forming a sloped triangle fan; similarly for "right side"
      - Simpler approach: divide footprint vertices into "left" and "right" of the ridge. Create two sloped planes: left vertices at topZ → ridge at ridgeZ, right vertices at topZ → ridge at ridgeZ.
      - Two gable triangles at the short ends (connect top ring vertices at each end to the ridge endpoint)
      - All triangles CCW from outside

   b. `buildHippedRoof(topRingXY, topZmm, holes, roofHeightMM)`:
      - Compute OBB of footprint
      - Ridge line: same as gabled but shorter (inset by shortAxis halfExtent from each end)
      - Ridge endpoints: OBB center +/- (halfExtents[0] - halfExtents[1]) * axes[0]
      - If building is nearly square (halfExtents[0] ~= halfExtents[1]), ridge degenerates to a point — fall back to pyramidal
      - Four sloped faces: two long-side trapezoids (connect long wall edges to ridge), two short-side triangles (connect short wall edges to nearest ridge endpoint)
      - All faces connect the top ring vertices at their wall height to the ridge line at ridgeZ

   c. `buildPyramidalRoof(topRingXY, topZmm, roofHeightMM)`:
      - Apex point: centroid of topRingXY at Z = max(topZmm) + roofHeightMM
      - For each consecutive pair of top ring vertices, create a triangle: v[i], v[i+1], apex
      - Triangle fan from apex to all perimeter edges
      - All triangles CCW from outside

3. Add `buildRoofForShape(shape: string, topRingXY, topZmm, holes, roofHeightMM): Float32Array`:
   - Dispatcher: 'flat' → buildFlatRoof, 'gabled' → buildGabledRoof, 'hipped' → buildHippedRoof, 'pyramidal' → buildPyramidalRoof
   - Unknown shapes → fall back to flat

4. Update `src/lib/buildings/buildingMesh.ts`:
   - Update `buildSingleBuilding()` to call `buildRoofForShape()` instead of always `buildFlatRoof()`
   - For non-flat roofs: the wall height = buildingHeightMM - roofHeightMM (walls go up to where the roof starts), roof adds the remaining height
   - If roofHeightMM >= buildingHeightMM, clamp wall height to buildingHeightMM * 0.5 (at least half walls)

5. Create `src/lib/buildings/__tests__/roof.test.ts`:

   - Test buildPyramidalRoof on square produces 4 triangles with shared apex
   - Test buildRoofForShape dispatches correctly for each shape string
   - Test buildRoofForShape falls back to flat for unknown shape 'onion'
