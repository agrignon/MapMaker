**Corrected approach:** Render `MobileViewToggle` as a **sibling** to the map and preview panes inside the `key="content"` div, not inside either pane. It should be positioned absolute with a high z-index so it floats above whichever pane is visible.

  {/* Map pane */}
  <div style={isMobile ? { ... } : { ... }}>
    {children}
  </div>
